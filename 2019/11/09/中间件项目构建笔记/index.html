中间件项目构建笔记

就近期搭建的中间件项目记录框架、环境以及踩过的坑

前端：

  • 基本框架：electron+vue.js
  • UI：iview

后端：

  • 基本语言：c++
  • 编译环境：cmake

前端记录

环境安装

第一步：安装必要的环境（ffi为不同语言间调用工具）

sudo npm install -g prebuild node-gyp electron
npm install ffi bindings --save

第二步：重新编译ref/ffi（c++端使用nan进行链接）


node-gyp rebuild -target=v7.1.0 -arch=x64 -dist-url=https://atom.io/download/atom-shell

运行流程：

  1. npm install
  2. node-gyp rebuild -target=v7.1.0 -arch=x64 -dist-url=https://atom.io/download/atom-shell
  3. npm start
